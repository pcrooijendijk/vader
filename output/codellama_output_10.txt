### Explanation:

The function `readfile()` is used to read a file from disk and output it to the HTTP response body.

This function does not check the path of the file, and will read any file with read permissions.

### Impact:

The function can be used to read any file on the server.

### Resolution:

Avoid using the `readfile()` function.
